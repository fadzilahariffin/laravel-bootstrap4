 How to install

- git clone https://github.com/fadzilahariffin/laravel-bootstrap4.git
- cp .env.example .env
- composer install
- php artisan key:generate
- setup DB connection
> DB_CONNECTION=mysql
> DB_HOST=127.0.0.1
> DB_PORT=3306
> DB_DATABASE=laravel-bootstrap4
> DB_USERNAME=root
> DB_PASSWORD=
- php artisan migrate
- npm install
- npm run dev
